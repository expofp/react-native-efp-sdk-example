# react-native-efp-sdk-example

Example app demonstrates how to use React Native packages [@expofp/react-native-efp-sdk](https://www.npmjs.com/package/@expofp/react-native-efp-sdk) and [@expofp/react-native-efp-crowdconnected](https://www.npmjs.com/package/@expofp/react-native-efp-crowdconnected)

## Quick start

```bash
git clone git@github.com:expofp/react-native-efp-sdk-example.git
cd ./react-native-efp-sdk-example

npm install

npm run expo:prebuild

# Android
npm run android
# iOS
npm run ios
```
